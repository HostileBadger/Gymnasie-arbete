What is SVG?

SVG is used to define graphics for the Web. I.e you create shapes with it.

You can easily create simple shapes like circles or squares with set commands:

Or you can create more complex shapes by defining the shape point by point:

Or (Shape borrowed from w3 schools)

How to use SVG

Creating a simple SVG, like a square for example, is as easy as creating an SVG tag with a width and a height.

<svg width="70" height="70">

<rect width="70" height="70" stroke="black" stroke-width="3" fill="blue"/>

</svg>

For the more advanced shapes you define each point individualy with their x,y coordinates.

<svg height="200" width="500">

<polyline fill="none"stroke="black"
points="10,20 80,25 60,40 110,10 20,140 200,100" />

</svg>
