how to use tables

to create the body for a table you use the <table> tag and then fill it with <tr> tags, that you then fill with <th>, or <td> tags

<table>
<tr class="row1">

<th class="header"> header column1</th>

<td class="column2"> column2</td>

<td class="column3"> column3</td>

</tr>
<tr class="row2">

<th class="header"> header column1</th>

<td class="column2"> column2</td>

<td class="column3"> column3</td>

</tr>
</table>
